使い方

地域の設定

お住まいの地域を選択することで、ゴミ収集日が自動的に更新されます。

ごみの日の表示

一番近いゴミの日から順番に、その日捨てるゴミの種類を表示します。

捨てるゴミの種類をタップすると、捨てることが可能なゴミの例を表示します。

詳細な分類は小金井市WEBのゴミ分別の手引きを参照してください。

このアプリについて

小金井市のゴミ出し日が一発で分かるアプリです。

運営について

このアプリはCode for Kanazawaが公開している5374.jpを基に、小金井市の個人が運営しています。

提供するゴミ情報について

ゴミ出しの日のデータは、小金井市が公開している「平成26年度ごみ・リサイクルカレンダー」PDFを元にして独自に実装しています。

ご使用における注意

このアプリはゴミ出しの日の確認が目的です。分別についてご自身でごみ・リサイクルカレンダーなどを確認のうえでご利用ください。

内容についての問合せについて

このサイトは個人が市民の利便性向上のために運営しており、行政とは無関係です。このサイトに関する質問や要望を小金井市や自治体、Code for Kanazawa様に問い合わせないでください。回収日や回収品目関する重大な誤りについてはhello@modshrink.comまでお知らせください。

Code for Kanazawaについて

コードで、世界をHappyに。
各地域には様々な課題があり、解決するためのコミュニティ（団体）も数多くあります。
課題の中には、ITやデザインの力で解決できるものも多くありますが、全てのコミュニティにそのスキル（技術）が備わっているわけではありません。

Code for Kanazawa（CfK）は、市民の課題を集め、その課題を整理・分析した上で、メンバーが実際に課題解決となるソフトウェアやハードウェア（仕組みや方法）を開発します。
私たちは、デザインをして、ソフトウェアコードを書き、課題を解決するサービスを完成させるのです。さらに、そのサービスを提供し続ける力も持ちます。

CfKは市民のための組織です。行政や民間企業の影響を受けずに中立・公益の立場から物事を判断するよう心がけています。
